## añadir assets
+ Crear una carpeta con el nombre assets
+ Añadir las imagenes
+ ir al archivo pubspec.yaml y buscar la linea:
> assets:
+ Descomentarla y agregar la direccion del asset en cuestion
<br>

        assets:
        - assets/img/pinguino.jpg
